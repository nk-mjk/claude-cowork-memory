## 위키 시스템

### 대화 시작 시 로딩

- 대화 시작 시 `wiki/INDEX.md`를 읽는다. (INDEX.md가 없으면 생략)
- INDEX.md에서 `*` 표시가 붙은 파일은 즉시 열어서 읽는다.
- 일반 항목은 지식 지도로만 파악하고, 관련 대화가 생길 때 열어본다.

### 노트 작성 트리거

- 의미 있는 작업 단위가 완료됐을 때 → "이 내용을 위키 노트로 남겨둘까요?" 라고 먼저 제안한다.
- 사용자가 "마무리", "정리해줘", "노트 남겨줘" 등 명시적으로 요청할 때 → 즉시 작성한다.
- 단순 작업·일회성 대화라면 노트를 만들지 않는다.

### 노트 파일 규칙

- 저장 위치: `wiki/`
- 파일명: `YYYYMMDD_주제.md`
- 내용: 그 세션의 지식·결정 1덩이 + 관련 노트로의 `[[링크]]`
- 작성 후 `wiki/INDEX.md` 지식 노트 섹션에 한 줄 설명을 추가한다.

### INDEX.md 한 줄 요약 작성 기준

- "무엇을 했다"보다 "무엇을 발견했다 / 결정했다"를 쓴다.
- 다음 세션의 Claude가 관련 노트를 열어볼지 판단하는 근거가 된다.

### 권한

- 새 노트 추가 및 INDEX.md 갱신 → 별도 허가 없이 진행한다.
- 기존 노트의 내용 수정·삭제 → 사용자 허가를 받은 후 진행한다.
- `about_me.md` 수정 → 사용자 허가를 받은 후 진행한다.
